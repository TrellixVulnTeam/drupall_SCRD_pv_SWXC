INTRODUCCIÓN
------------

El módulo de convocatorias es utilizado para implementar la conexión con el API 
de SICON proveyendo diferentes bloques y páginas de contenido cuya fuente es SICON.


INSTALACIÓN
------------

 * Es necesario instalar el módulo "Convocatorias" desde la administración


HERRAMIENTAS DISPONIBLES
-------------

* Tipo de bloque "Convocatorias recientes"
