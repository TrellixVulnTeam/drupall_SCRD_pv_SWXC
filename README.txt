Los estilos y el javascript son minificados a traves del gulp. 

## INICIALIZACIÓN
Si la carpeta ./node_modules no existe, correr el comando "npm install" en la raiz del proyecto

## MODIFICACIÓN
Se ejecuta el comando node "npm start" en la raiz del proyecto antes de realizar los cambios en el ./scss o el javascript de ./lib
Automaticamente los cambios son inyectados como librerias
